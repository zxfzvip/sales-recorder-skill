# 销售记账技能

## 触发条件

当用户提到以下关键词时激活：
- 记账、录入、拿货、记录
- "记一下"、"记一笔"、"记个账"
- 快递相关（快递数量、快递费）

## 参数提取

从用户消息中提取：
- **商品名称**：如"风流果"、"润滑油"等
- **数量**：支持"数量30"、"30个"等格式
- **单价**：支持"价格5"、"5元"等格式
- **快递数量**：支持"快递10"等
- **快递费**：支持"价格3"等
- **文件名**：用户指定，如"弟弟"、"央央"、"薛泽凯"

## 执行方式

1. 解析用户消息中的各项参数
2. 使用 exec 执行 Python 脚本：
   ```bash
   python3 ~/.openclaw/skills/sales-recorder-skill/scripts/record.py <target> <product> <qty> <price> <expr_count> <expr_price>
   ```
3. 返回执行结果（包含计算后金额）

## 示例

用户说："风流果数量30价格5.5快递10价格3 弟弟 记一下"

解析参数：
- target: 弟弟
- product: 风流果
- qty: 30
- price: 5.5
- expr_count: 10
- expr_price: 3

执行命令：
```bash
python3 ~/.openclaw/skills/sales-recorder-skill/scripts/record.py 弟弟 风流果 30 5.5 10 3
```

返回：
```
✅ 已添加到 弟弟.xlsx 第10行！

| 商品 | 数量 | 单价 | 小计 |
|------|------|------|------|
| 风流果 | 30 | 5.5 | ¥165.00 |

| 快递单数 | 快递价格 | 小计 |
|----------|----------|------|
| 10 | 3 | ¥30.00 |
```
